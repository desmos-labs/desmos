# Query posts module parameters
This query endpoint returns all the parameters of the `posts` module.

**CLI**
 ```bash
desmoscli query posts parameters
```

**REST**
```
/posts/parameters

# Example
# curl http://lcd.morpheus.desmos.network:1317/posts/parameters
```
