# Query posts module parameters
This query endpoint returns all the parameters of the `posts` module.

**CLI**
 ```bash
desmoscli query posts params

# Example
# desmoscli query posts params
``` 
**REST**
```
/posts/params

# Example
# curl http://lcd.morpheus.desmos.network:1317/posts/params
```
