Boolean expression to compare columns of type "jsonb". All fields are combined with logical 'AND'.

```graphql
input jsonb_comparison_exp {
  _contained_in: jsonb
  _contains: jsonb
  _eq: jsonb
  _gt: jsonb
  _gte: jsonb
  _has_key: String
  _has_keys_all: [String!]
  _has_keys_any: [String!]
  _in: [jsonb!]
  _is_null: Boolean
  _lt: jsonb
  _lte: jsonb
  _neq: jsonb
  _nin: [jsonb!]
}
```
